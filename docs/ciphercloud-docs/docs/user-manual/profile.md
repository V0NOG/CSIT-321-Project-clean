# Managing Your Profile

## Viewing Your Profile

Click **Profile** in the left sidebar, or click your name/avatar in the top header.

Your profile displays:
- First and last name
- Email address
- Phone number (optional)
- Bio (optional)
- Address details (optional)
- Two-factor authentication status

---

## Updating Personal Details

1. On the Profile page, click **Edit** next to the section you want to update.
2. Make your changes.
3. Click **Save**.

### Editable Fields

| Field | Notes |
|-------|-------|
| First Name | Your given name |
| Last Name | Your family name |
| Phone | Optional contact number |
| Bio | A short description about yourself |
| Country | Your country of residence |
| City / State | City or state |
| Postal Code | Postcode or ZIP code |

:::info Email address
Your email address is your login identity and cannot be changed after registration in the current version.
:::

---

## Two-Factor Authentication Status

Your profile page shows whether 2FA is currently enabled or disabled. Click the link to go directly to the [Two-Factor Authentication](./two-factor-auth) setup page.

---

## Account Security Tips

- Use a strong, unique password for your Cipher Cloud account
- Enable two-factor authentication
- Sign out when using shared computers
- Keep your authenticator app backed up
