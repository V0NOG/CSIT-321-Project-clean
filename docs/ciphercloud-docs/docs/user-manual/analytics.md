# Analytics Dashboard

The Analytics page shows statistics about your Cipher Cloud usage storage consumed, file counts, upload trends, and activity patterns.

---

## Accessing Analytics

Click **Analytics** in the left sidebar.

---

## What Is Displayed

### KPI Tiles (top row)

| Tile | What it shows |
|------|--------------|
| Total Files | The number of files you have uploaded |
| Total Storage Used | Combined size of all your encrypted files |
| Total Sessions | Number of times you have logged in |
| Total Shares | Number of share invitations you have sent |

### Upload Trend (last 30 days)

A bar chart showing how many files you uploaded on each day over the past 30 days. Useful for tracking your usage patterns over time.

### Top Actions

A breakdown of the most common actions recorded in your account over the past 30 days (uploads, downloads, shares, etc.).

### Storage by File Type

A chart showing your storage usage broken down by file category:

| Category | Examples |
|----------|---------|
| Document | PDF, Word, Excel, PowerPoint |
| Image | JPG, PNG, GIF, HEIC, WebP |
| Video | MP4, MOV, AVI, MKV |
| Audio | MP3, WAV, FLAC, AAC |
| Archive | ZIP, RAR, 7z, TAR |
| Other | All other file types |

---

## Privacy Note

Analytics data is calculated from your own file metadata filenames, sizes, and MIME types. Your file **content** is never read or analysed by the server.
