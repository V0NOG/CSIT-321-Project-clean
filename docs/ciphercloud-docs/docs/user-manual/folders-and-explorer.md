# Folders & File Explorer

The Explorer page is the primary interface for managing your files. It combines a folder tree on the left with a file grid on the right.

---

## Creating a Folder

1. Click the **+** icon at the top of the Folders panel, **or** right click empty space in the folder panel and select **New Folder**.
2. Type the folder name.
3. Press **Enter** to confirm or **Escape** to cancel.

---

## Renaming a Folder

- **Right-click** the folder then select **Rename**, or
- **Double-click** the folder name

Type the new name and press **Enter**.

---

## Deleting a Folder

1. Right click the folder then select **Delete Folder**.
2. Confirm the deletion.

:::info Files are not deleted
Deleting a folder moves its files back to **All Files** (root level). No files are lost.
:::

---

## Moving Files to a Folder

### Drag and Drop

1. Click and hold a file card in the grid.
2. Drag it onto a folder name in the left panel the folder highlights when it is a valid drop target.
3. Release to move the file.

You can also drag a file onto **All Files** to move it back to the root level.

### Right-Click Menu

1. Right click a file then select **Move to…**
2. A panel lists all your folders. Click the destination folder.

---

## Right-Click Context Menu

Right clicking opens a context menu with actions depending on what you clicked:

**On a file:**
| Action | Result |
|--------|--------|
| Download | Decrypt and download the file |
| Rename | Edit the file name inline |
| Move to… | Move to a different folder |
| Share | Open the share dialog |
| Delete | Permanently delete the file |

**On a folder (in sidebar):**
| Action | Result |
|--------|--------|
| Rename | Edit the folder name inline |
| Delete Folder | Delete the folder (files moved to root) |

**On empty space in file grid:**
| Action | Result |
|--------|--------|
| Upload File | Open the file picker |
| New Folder | Create a new folder |

---

## Keyboard Shortcuts in Inline Edit Mode

| Key | Action |
|-----|--------|
| Enter | Confirm the new name |
| Escape | Cancel and revert to the original name |
